
📋 PROJECT OVERVIEW
Project Name: LANDGUARD
Client: Government of Ghana
Purpose: National land registry and marketplace platform to eliminate land fraud by providing a verified, map-based system showing land ownership status, availability, and property details across Ghana.
Target Platforms: iOS & Android (Cross-Platform), website
Deployment: cloud infrastructure with high-security compliance
Assistance: Help button and a chatbot that learns and underdstand everything about Languard and assist users


🛠️ RECOMMENDED TECH STACK

 **Frontend (Mobile App)**

✅ Framework: Flutter (Dart) 
   - Reason: Superior UI consistency, enterprise-grade scalability, near-native performance, and better long-term maintainability for government applications [[9]][[11]][[12]]
   - Alternative: React Native only if existing JS team is available

✅ UI/UX Design System: Glassmorphism + 3D Elements
   - Packages: flutter_glass_morphism [[28]], glassmorphic_ui_kit [[31]]
   - Animation: Flutter Animate + Rive for 3D micro-interactions
   - Theming: Dynamic light/dark mode

✅ Mapping Integration:
   - google_maps_flutter: For interactive Ghana map visualization
   - ghana_post_gps_api: Custom wrapper for GhanaPostGPS API integration [[1]][[2]][[6]]
   - geolocator: For user location services
   - polygon_drawing: Allow sellers to mark land boundaries on map

✅ State Management: Riverpod or Bloc (for scalable, testable architecture)
✅ Local Storage: Hive (for offline map caching)
✅ API Client: Dio with retry logic & encryption
```

### **Backend (Server)**
```
✅ Runtime: Node.js with Express.js
   - Recommendation: Node.js for real-time features + JavaScript ecosystem synergy

✅ Database: MongoDB Atlas (Government-tier cluster)
   - Geospatial Indexing: 2dsphere indexes for land parcel queries [[19]][[25]]
   - GeoJSON format for all location data (longitude first, then latitude) [[19]]
   - Schema design with GeoJSON Polygon for land boundaries

✅ Authentication: 
   - JWT + Refresh Tokens with RSA256 signing
   - Biometric login support (FaceID/Fingerprint)

✅ File Storage:

✅ Real-time Features:

## 🗄️ MONGODB DATABASE SCHEMA (Key Collections)

```javascript
// users collection
{
  _id: ObjectId,
  role: "buyer" | "seller" | "government_admin",
  personalInfo: {
    fullName: String,
    ghanaCardNumber: { type: String, unique: true, encrypted: true },
    phoneNumber: { type: String, verified: Boolean },
    email: String,
    profilePhotoUrl: String
  },
  sellerInfo: { // conditional
    businessRegNumber: String,
    tin: String,
    bankAccount: { encrypted: true },
    verificationStatus: "pending" | "verified" | "rejected",
    verifiedBy: ObjectId, // gov admin ID
    verifiedAt: Date
  },
  location: {
    type: "Point",
    coordinates: [longitude, latitude] // GeoJSON [[19]]
  },
  createdAt: Date,
  lastLogin: Date,
  isSuspended: Boolean
}

// landParcels collection
{
  _id: ObjectId,
  ownerId: ObjectId, // reference to users
  status: "available" | "under_offer" | "sold" | "disputed",
  propertyType: "vacant_land" | "with_building" | "commercial" | "residential",
  
  // Geospatial data - CRITICAL [[19]][[25]]
  geometry: {
    type: "Polygon", // or "Point" for small plots
    coordinates: [[ [lon, lat], [lon, lat], ... ]] // closed loop
  },
  ghanaPostGPSAddress: String, // e.g., "GA-123-4567"
  
  // Property details
  details: {
    sizeSqM: Number,
    sizeAcres: Number,
    priceGHS: Number,
    negotiable: Boolean,
    description: String,
    features: [String] // ["water", "electricity", "fenced", ...]
  },
  
  // Media
  media: [{
    url: String,
    type: "image" | "video" | "360_tour",
    uploadedAt: Date,
    verified: Boolean // admin-approved
  }],
  
  // Documents (seller-uploaded proofs)
  documents: [{
    type: "title_deed" | "site_plan" | "ghana_card" | "utility_bill",
    fileUrl: String, // encrypted S3 path
    uploadedAt: Date,
    verifiedBy: ObjectId,
    verifiedAt: Date,
    expiryDate: Date // for time-bound docs
  }],
  
  // Transaction trail
  history: [{
    action: "listed" | "price_changed" | "status_updated" | "sold",
    performedBy: ObjectId,
    timestamp: Date,
    metadata: {} // e.g., { oldPrice, newPrice }
  }],
  
  // Indexes
  createdAt: Date,
  updatedAt: Date,
  isFeatured: Boolean
}

// Create geospatial index [[19]][[25]]
db.landParcels.createIndex({ geometry: "2dsphere" })
db.landParcels.createIndex({ "ghanaPostGPSAddress": "text" })
db.landParcels.createIndex({ status: 1, createdAt: -1 }) // for feed queries

// verificationQueue collection (for admin workflow)
{
  _id: ObjectId,
  sellerId: ObjectId,
  documentsSubmitted: [{
    type: String,
    fileUrl: String,
    ocrExtractedData: {} // auto-extracted metadata
  }],
  status: "pending" | "approved" | "rejected" | "more_info_needed",
  assignedOfficer: ObjectId,
  reviewNotes: String,
  submittedAt: Date,
  reviewedAt: Date
}

// messages collection (encrypted)
{
  _id: ObjectId,
  conversationId: String, // unique per buyer-seller pair
  senderId: ObjectId,
  receiverId: ObjectId,
  landParcelId: ObjectId, // context
  content: { encrypted: true }, // end-to-end encryption
  attachments: [{ url: String, type: String }],
  sentAt: Date,
  readAt: Date,
  flagged: Boolean // for moderation
}
```

## 🔐 SECURITY & COMPLIANCE (Government-Level)

```
✅ Data Encryption:
   - TLS 1.3 for all API traffic
   - AES-256 encryption for sensitive fields (Ghana Card, bank details)
   - MongoDB field-level encryption for PII

✅ Authentication & Authorization:
   - Multi-factor authentication (SMS + biometric) for sellers/buyers/admins
   - Role-based access control (RBAC) with least-privilege principle
   - Session management with automatic timeout

✅ Document Security:
   - Watermark all displayed documents with user ID + timestamp
   - Disable right-click/download on web views
   - OCR extraction runs in isolated sandbox

✅ Audit & Monitoring:
   - Immutable audit logs (write-once storage)
   - Real-time fraud detection: flag duplicate GPS coordinates, rapid price changes
   - Integration with Ghana Cyber Security Authority protocols

✅ Compliance:
   - Ghana Data Protection Act, 2012 (Act 843)
   - Lands Commission regulations
   - GDPR principles for data minimization & user rights
   - Regular third-party security audits

✅ Disaster Recovery:
   - Multi-region MongoDB Atlas deployment
   - Daily encrypted backups with 30-day retention
   - RTO < 4 hours, RPO < 15 minutes
```
